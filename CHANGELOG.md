# Changelog

All notable changes to ArcaAI are recorded here, organised by Implementation Pack release.

The format is loosely based on [Keep a Changelog](https://keepachangelog.com/).

This project adheres to **Implementation Pack versioning** — each release is a tagged, immutable snapshot of the canonical specifications at a point in time. See `CONTRIBUTING.md` for the release mechanics.

---

## [Unreleased]

### Added

- Initial repository scaffold
- Three foundational ADRs ratified:
  - ADR-0001: Pre-trained model positioning (reference models, not production)
  - ADR-0002: Three-stage model lifecycle (Reference → Upskilling → Continuous Improvement)
  - ADR-0003: Pipeline-as-platform positioning
- Specification template with sister-specifications discipline
- ADR and RFC templates and processes
- SME panel composition and review protocols
- Document register
- Glossary stub
- START_HERE.md — navigation map for new readers, with role-based entry points and topic-to-document mapping
- SESSION_NOTES.md — append-only chronological record of reasoning behind decisions across chat sessions
- SESSION_PROTOCOLS.md — formal opening and closure rituals for every chat session, with continuity-risk mitigations

### Status

No Implementation Pack has yet been released. The first release (`pack-v1.0.0`) is targeted for end of Month 5 of the design phase, contingent on all eight specifications reaching ratified status.
